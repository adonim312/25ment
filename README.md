## Overview
詳しいことは[本家](https://github.com/chimerast/niconico-speenya)を参照してください。
**25ment**はFGMF用にカスタマイズされたもので、フォント、文字色、文字の大きさ、文字のスピード、スタンプの種類や数などがカスタマイズされています。

また、本家にない追加機能として、スマートフォン画面をサイリウムの代わりにする**スマリウム**機能があります。
